# Habit Tracker Pro

A minimalist habit tracking web app built with vanilla HTML, CSS, and JavaScript.

## Features
- Drag & drop reordering
- Categories
- Start & end dates
- Dark mode
- Persistent local storage
- GitHub Pages deployment

## Tech Stack
- HTML
- CSS (custom properties for theming)
- JavaScript
- Git & GitHub
